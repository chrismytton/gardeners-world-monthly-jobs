Gardener's World monthly jobs API

jobs.json

API of what to do in the garden.

The API returns an object where the keys are the month names, January through to December. Each key returns an object with 3 keys, Flowers, Fruit and veg, Greenhouse and Garden maintenance which correspond to different areas of the garden. These keys returns arrays of strings. Each string is a single job that needs doing in the current month for the corresponding area of the garden.
